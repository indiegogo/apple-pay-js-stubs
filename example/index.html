Instructions

Initially this page is loaded without apple-pay-js-stubs loaded. To try out apple-pay-js-stubs you can control the stub behaviour using the Example Console above.

You can enable stubbing for a given configuration by:

  1. Clicking 'Load apple-pay-js-stubs'
  2. Clicking your preferred configuration option
  3. then click the '🍎 Pay'
The 'Log of Stubbed Behaviour' outputs the stubbed user behaviour as a result of your clck on the '🍎 Pay' button.
Note: after every '🍎 Pay' press the stub behaviour will be cleared. So please reselect another option between presses.
